# FAQ - Часто задаваемые вопросы

## Общие вопросы

### Q: Что делает этот мод?
**A:** Мод позволяет контролировать направление кнокбека (отбрасывания) при атаке врага. Перед ударом голова персонажа автоматически поворачивается на заданный угол, что изменяет направление отбрасывания цели.

### Q: Для какой версии Minecraft этот мод?
**A:** Мод создан для Minecraft 1.21.1 на Fabric.

### Q: Можно ли использовать на Forge?
**A:** Нет, это мод для Fabric. Для Forge нужна полная переработка кода.

### Q: Работает ли на серверах?
**A:** Технически может работать, но:
- Может нарушать правила сервера
- Может быть определён античитом как читерство
- **Не рекомендуется для использования на серверах**
- Предназначен для одиночной игры и обучения

## Сборка

### Q: Как собрать мод без ПК, только онлайн?
**A:** Используйте Replit:
1. Создайте Java Repl на replit.com
2. Загрузите все файлы проекта
3. Выполните `gradle wrapper --gradle-version 8.5`
4. Запустите `./gradlew build`
5. Скачайте готовый .jar из `build/libs/`

### Q: Сколько времени занимает сборка?
**A:** 
- Первая сборка: 10-20 минут (скачивание Minecraft и зависимостей)
- Последующие сборки: 1-3 минуты

### Q: Почему сборка так долго идёт?
**A:** Gradle должен скачать:
- Minecraft 1.21.1 (~150 MB)
- Fabric API и все библиотеки
- Маппинги Yarn для декомпиляции
Это нормально для первой сборки.

### Q: Сборка упала с ошибкой "Out of memory"
**A:** Добавьте в `gradle.properties`:
```properties
org.gradle.jvmargs=-Xmx3G
org.gradle.daemon=false
```

### Q: Gradle не может скачать зависимости
**A:** Проблема с интернетом или репозиториями. Попробуйте:
```bash
./gradlew build --refresh-dependencies
```

## Установка

### Q: Как установить мод?
**A:**
1. Установите Fabric Loader для Minecraft 1.21.1
2. Скачайте Fabric API 0.86.0+1.21.1
3. (Опционально) Скачайте Mod Menu 11.0.1+
4. Поместите все .jar файлы в папку `mods`
5. Запустите Minecraft

### Q: Где находится папка mods?
**A:**
- **Windows:** `%appdata%\.minecraft\mods`
- **Mac:** `~/Library/Application Support/minecraft/mods`
- **Linux:** `~/.minecraft/mods`

### Q: Мод не загружается, что делать?
**A:** Проверьте:
1. Установлен ли Fabric Loader для версии 1.21.1
2. Установлен ли Fabric API в папке mods
3. Версия Java должна быть 21 или выше
4. Посмотрите логи в `logs/latest.log` на ошибки

### Q: Нужен ли Mod Menu?
**A:** Нет, но рекомендуется. Без Mod Menu:
- Мод будет работать
- Но не будет доступа к GUI настроек
- Настройки можно изменить только в коде

## Использование

### Q: Как активировать мод?
**A:** Нажмите клавишу **R** (по умолчанию). Повторное нажатие - выключение.

### Q: Как изменить клавишу активации?
**A:** 
**Способ 1 (в игре):**
- Options → Controls → Key Binds
- Найдите "Toggle Knockback Trainer"
- Измените клавишу

**Способ 2 (в коде):**
- Откройте `ModKeybinds.java`
- Измените `GLFW.GLFW_KEY_R` на другую клавишу
- Пересоберите мод

### Q: Как открыть настройки?
**A:**
1. Установите Mod Menu
2. Main Menu → Mods
3. Найдите "Knockback Trainer"
4. Нажмите кнопку настроек (шестерёнка)

### Q: Что такое Yaw Offset?
**A:** Угол поворота головы перед ударом в градусах:
- **0°** - без изменений
- **45°** - отбрасывание вправо
- **-45°** - отбрасывание влево
- **90°** - отбрасывание сильно вправо
- **180°** - отбрасывание назад

### Q: Что такое Delay Ticks?
**A:** Задержка между поворотом головы и ударом:
- **0 тиков** - мгновенный удар
- **2 тика** - ~100мс задержка (по умолчанию)
- **10 тиков** - ~500мс задержка
- **20 тиков** - 1 секунда

1 тик = 50 миллисекунд = 1/20 секунды

### Q: Мод не работает, хотя активирован
**A:** Проверьте:
1. Действительно ли мод включён (проверьте логи)
2. Правильно ли вы целитесь в врага
3. Delay Ticks не слишком большой
4. Нет конфликтов с другими модами

## Настройка

### Q: Как сохранить настройки между сессиями?
**A:** В текущей версии настройки не сохраняются автоматически. Они сбрасываются при перезапуске Minecraft. 

Для постоянных настроек измените значения по умолчанию в `KnockbackConfig.java`:
```java
private static float yawOffset = 90.0f;  // Ваше значение
private static int delayTicks = 5;        // Ваше значение
```

### Q: Можно ли добавить автосохранение?
**A:** Да, можно добавить сохранение в JSON файл. Пример кода будет в будущих версиях.

### Q: Какие оптимальные настройки?
**A:** Зависит от цели:
- **Для комбо:** Yaw=45°, Delay=1
- **Для защиты:** Yaw=-90°, Delay=0
- **Для практики:** Yaw=60°, Delay=2

## Проблемы и решения

### Q: Minecraft крашится при запуске с модом
**A:** Возможные причины:
1. **Несовместимая версия Minecraft** - должна быть 1.21.1
2. **Старая версия Java** - обновите до Java 21
3. **Отсутствует Fabric API** - установите его
4. **Повреждённый .jar файл** - перескачайте

Проверьте crash-report в `crash-reports/`

### Q: Файл .jar называется "KnockbackTrainer-1.0.0 (1).jar"
**A:** Браузер добавил `(1)` при повторном скачивании. Просто переименуйте файл, удалив ` (1)`.

### Q: Размер .jar файла подозрительно маленький/большой
**A:** Нормальный размер: **50-150 KB**
- Если <30 KB: сборка неполная, пересоберите
- Если >10 MB: возможно включены лишние зависимости

### Q: GUI настроек не открывается
**A:** Установите Mod Menu 11.0.1+. Без него GUI недоступен.

### Q: Конфликт с другими модами
**A:** Мод может конфликтовать с:
- Другими модами управления атакой
- Некоторыми PvP модами
- Читерскими клиентами

Попробуйте отключить другие моды по одному.

### Q: На сервере мод не работает / я получил бан
**A:** 
- Мод клиентский, сервер его не видит
- Но античит может засечь подозрительное поведение
- **НЕ используйте на серверах с античитом**
- Используйте только в одиночной игре

## Разработка

### Q: Могу ли я модифицировать код?
**A:** Да, мод под лицензией MIT. Вы можете:
- Изменять код
- Распространять изменённые версии
- Использовать в своих проектах
- При условии сохранения оригинальной лицензии

### Q: Как добавить новые функции?
**A:** Примеры расширений:
1. **Автосохранение** - добавьте JSON сериализацию
2. **HUD индикатор** - используйте RenderSystem
3. **Звуковые эффекты** - добавьте SoundEvents
4. **Particle effects** - используйте ParticleManager

### Q: Как отладить мод?
**A:**
```bash
./gradlew runClient
```
Это запустит Minecraft с вашим модом в dev-окружении.

### Q: Можно ли портировать на другие версии?
**A:** Да, но потребуется:
- Обновить зависимости в build.gradle
- Возможно изменить API вызовы (они могут отличаться)
- Протестировать функциональность

### Q: Как добавить поддержку других языков?
**A:** Создайте файлы в `assets/knockbacktrainer/lang/`:
- `de_de.json` - немецкий
- `fr_fr.json` - французский
- `es_es.json` - испанский
И т.д.

## Технические детали

### Q: Почему именно Fabric, а не Forge?
**A:** 
- Fabric легче и быстрее
- Более современный API
- Лучше подходит для клиентских модов
- Быстрее обновляется на новые версии

### Q: Как мод влияет на производительность?
**A:** Минимально:
- Код выполняется только при включённом моде
- Проверки каждый тик занимают <0.1ms
- Нет влияния на FPS

### Q: Безопасен ли мод?
**A:** Да:
- Открытый исходный код
- Не собирает данные
- Не модифицирует игровые файлы
- Не взаимодействует с сетью

### Q: Можно ли использовать вместе с Optifine?
**A:** 
- Optifine несовместим с Fabric
- Используйте вместо него: Sodium + Iris + Mod Menu
